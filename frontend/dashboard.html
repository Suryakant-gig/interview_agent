Self Correcting RAG Workflow

START

User asks a question.

↓

decide_retrieval

Decide whether retrieval is needed or direct generation is enough.

↓

retrieve()

Retrieve relevant documents from vector database.

docs = retriever.invoke(question)
↓

grade_documents()

Check whether retrieved documents are relevant to the question.

score = doc_grader.invoke({ "question": question, "documents": docs_text })
↓

generate()

Generate grounded answer using retrieved context.

generation = generate_chain.invoke({ "question": question, "context": context })
↓

Hallucination Check

Validate whether answer is supported by retrieved documents.

↓

rewrite_query()

Rewrite query if retrieval quality is poor.

better_question = rewrite_chain.invoke({ "question": question })
↓

FINAL ANSWER

Return grounded and validated response.
